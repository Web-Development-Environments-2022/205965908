About Me






זאב נולד בחיפה להוריו אשר וכרמלה ומגיל צעיר עברה משפחתו לגור בצפת.
הוא היה מאוד שובב, הוא מאוד להשתולל. היה לו מזל, מכיוון שבית החולים שכן מעבר לכביש, כמעט ולא היה שבוע שלא ביקר שם.
זכרונו הראשון מהילדות הוא שבגיל שנתיים וחצי קיבל את האופניים הראשונים שלו. סיפור ידוע בצפת היה כשהוא נפל מקומה רביעית בבניין שבו גר, ושכב כמה ימים מחוסר הכרה בבית החולים.
בגיל 10 עברה משפחתו לנתניה, עקב התמנות אביו למנהל חברת הטלפונים האזורי.
זאב למד בחטיבת ביניים "רופין" ובחטיבה עליונה "אורט יד-ליבוביץ".
הוא שירת בצבא בספינת טילים בחיל הים והשתתף בכל מיני מבצעים סודיים ידועים יותר וידועים פחות.
זאב למד באוניברסיטת בר-אילן ומאז הוא עובד כמנהל לוגיסטיקה בעיריית נתניה.

זאב אוהב לקרוא ספרים, ובעיקר את הספרים של ג'ול וורן (וביניהם: מסביב לעולם ב-80 יום, 20,000 מ"ל מתחת למים ועוד). כמו כן, זאב אוהד את קבוצת הכדורגל מבי נתניה, ואת השחקן לשעבר עודד מכנס.
בתחום המוזיקה הוא העריץ מאוד את להקת "הגשש החיוור" ואת החיפושיות, וכמובן היו לו כמעט את כל הדיסקים שלהם.
זאב אוהב לאכול תירס, ולעומת זאת שונא לאכול פיצה. בשעות הפנאי שלו אוהב לפתור תשבצים וסודוקו בעיתון, לצפות בטלווזיה במשחקי כדורגל.

לזאב יש שלושה נכדים מתוקים!

Q&A

1) ההבדלים בין Web1, Web2, Web3:

כדי להסביר את ההבדלים נסביר קודם על כל Web בנפרד:

Web 1: אחת מהגרסאות העיקריות של האינטרנט כפי שאנו מכירים אותה. בגרסה זו, למשתמשי האינטרטנט הייתה אינטראקציה מוגבלת בינם לבין האתר עצמו. כמו כן, אתרי אינטרנט כמעט ולא קיימו אינטראקציה זה עם זה. משתמשים לא יכלו לפרסם או להגיב משוב. Web 1 הייתה דרך טובה לתת לעסקים ליידע אנשים שהם נמצאים שם ולהציג מידע על העסק שלהם.

Web 2: בגרסה זו של האינטרנט משתמשים יכול ליצור אינטראקציה עם האתר עצמו - הם יכלו להגיב, לפרסם ולהשאיר משוב. Web 2 עודד השתתפות, שיתוף פעולה ושיתוף מידע ממשתמשי האינטרנט. בגרסת אינטרנט זו, המשתמשים נחשפו לאתרים כגון פייסבוק, יוטיוב וויקיפדיה. שימוש בWeb 2 אפשר לאנשים לתקשר אחד עם השני ולשתף חוויות מחיי היום יום שלהם. גרסה זו היא האינטרנט כפי שרוב האנשים מכירים אותה. אנשים יכלו כעת גם לחפש את מה שהם רוצים לראות באמצעות מנועי חיפוש. דבר נוסף שמאפיין את Web 2 הוא דפדפני האינטרנט. חברות התחילו ליצור דפדפני אינטרנט כגון גוגל כרום, אינטרנט אקספלורר ועוד רבים אחרים. בניגוד ל Web 1, שנחשב לתוכן "קריאה" בלבד, Web 2 נחשב לתוכן "קריאה-כתיבה".

Web 3: אז מה עוד אנחנו יכולים לבקש שלא מיושם בWeb 3 ?Web 2 עוסק בלתרגם מידע כמו שבני אדם יכולים. אם נפנה ל Web 1 כתוכן "קריאה" ול Web 2 כתוכן "קריאה-כתיבה", נוכל להתייחס ל Web 3 כתוכן "קריאה-כתיבה-ביצוע". אנחנו רוצים שהמכונה תפעל לבד - המחשב יתנהג כאילו הוא עוזר אישי. ה Web 3 היא גרסת האינטרנטשבה אנו משתמשים כיום.

נסכם את ההבדלים בין Web 2 לבין Web 3 לפי קטגוריות:

חיפוש - החיפוש בWeb 2 הוא לפי מילות מפתח בעוד שבWeb 3 הוא לפי קישוריות.
התמקדות - ההתמקדות בWeb 2 זה בחברה ובWeb 3 בכל אדם באופן אישי.
אינפורמציה - האינפורמציה בWeb 2 היא דינאמית בעוד שבWeb 3 היא אישית.





2) הסבר על ההבדלים בין Web 1-3 ע"י דוגמא:

נניח שאתה רוצה להזמין תור לרופא הפרטי שלך דרך האתר שלו. בWeb 1 תהיה לך את היכולת לצפות רק במה שהרופא שלך החליט לשים באתר שלו. הוא כנראה ישים את הכתובת ומספר הטלפון שלו באתר ואתה תצטרך להתקשר אליו כדי לשנות דברים בפגישה איתו, כמו כן תצטרך לדעת את כתובת האתר שלו.

בWeb 2 אתה יכול לחפש "רופא פרטי", ואחרי כמה חיפושי אתרים תוכל למצוא את האתר של הרופא הפרטי שלך. אחרי שתכנס לאתר שלו תוכל לראות את מתי הוא זמין בלוח הפגישות ולקבוע פגישה בעצמך.

בWeb 3 תוכל לבקש מהטלפון שלך (סירי באייפון) לקבוע תור לרופא הפרטי שלך בזמן שמתאים לך, והטלפון יעשה את זה ויודיע לך אם הוא עשה את זה בהצלחה!
